# JSOC

## What does "JSOC" mean?

JSOC stands for JSON Structure Oriented Components. It is an approach to build dynamic UI components which are suitable for representing JSON type of data.
These components self configure based on the JSON data and don't need to manually reconfigured every time the JSON data changes.

There are several types of components suitable for representing JSON data depending on the shape/structure such as DataGrid/DataTable, Tree, Chart, etc. 
These components need manual configuration and you need to know the exact shape of the data beforehand to generate those configurations.
JSOC is an approach to build dynamic configurations for these components so that they can be self configured based on the JSON data.

## What is JSOC Grid then?
DataGrid or DataTable are most widely used components for representating JSON data.

JSOC Grid is a sub solution under the JSOC umbrella. 
JSOC Grid is an approach and set of libraries that help to generate and manage dynamic configurations for popular DataGrid/DataTable components based on the JSON data. 

### Future 
In the future we are planning to add support for other types of JSOC components such as Tree, Chart, etc.
