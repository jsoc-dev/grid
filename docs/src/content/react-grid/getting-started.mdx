### Installation

- Install [@jsoc/react-grid](https://www.npmjs.com/package/@jsoc/react-grid) and the plugin package for your selected grid library:

  ```sh npm2yarn
  npm install @jsoc/react-grid ${plugin.packageName}
  ```

- Install the peer dependencies:
  - `react`
  - `react-dom`
  - <code><SelectedPlugin peerDeps/></code>

> [!TIP]
>
> See the [Plugins](/react-grid/plugins) page for package names. Refer to each package's `package.json` on [GitHub](https://github.com/jsoc-dev/grid/tree/main/packages) for exact peer dependency ranges.

#### Import necessary features

From `@jsoc/react-grid`:

- `GridStoreProvider`
- `useGridStoreSelector`

From your plugin package:

- `useGridStore`

```typescript
import { GridStoreProvider, useGridStoreSelector } from "@jsoc/react-grid";
import { useGridStore } from "${plugin.packageName}";
```

> [!TIP]
>
> You can refer to the [Plugins](/react-grid/plugins) page for import paths for other grid libraries.

#### Create the store

Pass your grid data directly to `useGridStore`, then read the active schema with `useGridStoreSelector`:

```typescript
const data = [{ id: 1, name: "Alice" }];

const gridStore = useGridStore(data);
const activeSchema = useGridStoreSelector(gridStore, (store) =>
  store.getActiveSchema(),
);
```

#### Wrap with `GridStoreProvider`

Wrap your grid component with `GridStoreProvider` and pass the store. Spread `activeSchema.config` into your grid library component:

```tsx
<GridStoreProvider value={gridStore}>
  <YourGridComponent key={activeSchema.id} {...activeSchema.config} />
</GridStoreProvider>
```

Replace `YourGridComponent` with the component from your grid library (for example, AG Grid's `AgGridReact`, Ant Design's `Table`, or TanStack Table's `useReactTable`).
