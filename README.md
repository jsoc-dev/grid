# jsoc/grid

JSOC Grid brings the JSOC (JSON Structure Oriented Components) philosophy to data grids, transforming raw JSON data into fully functional grid configurations at runtime.

With support for multiple popular grid libraries, it enables a plug-and-play architecture where grid configurations are automatically generated based on the JSON data.
